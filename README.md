# CylancePROTECT-Samples
This repo contains sample applications making use of CylancePROTECT REST APIs and Syslogs from the CylancePROTECT

| **Sample Name** | **REST / Syslog Sample** | **Tech Stack** |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ----------------------- |
|CylancePROTECT React Native Sample| **Both:** A REST API Focused App with a simple Syslog Forwarding/Notification server| **React-Native-App:** React Native, Redux, Javascript.  **Notification/Syslog Server:** NodeJS/Firebase  |
|CylancePROTECT C#/.Net WPF Sample | **Both:** Contains CylancePROTECT REST API Examples, and a Syslog client example | C#, .Net, Windows Presentation Foundation (WPF) in Visual Studio.
